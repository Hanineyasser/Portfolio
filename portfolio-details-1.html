Gym Membership Description

This project is a comprehensive backend system for a gym management application, built exclusively using Java. The system features a robust, monolithic architecture designed for scalability and handles all server-side logic. Key functionalities include secure member authentication, role-based access control, membership plan and subscription management, class scheduling, and attendance tracking. This project showcases a complete, enterprise-level backend solution developed entirely within the Java ecosystem.
