Social Media Platform Description

This project is a desktop social media application built entirely in Java, featuring a client-server architecture. The intuitive graphical user interface (GUI) was developed with Java Swing, while the backend handles all core logic and data management with Java. The platform supports essential social media functions, including secure user registration, profile customization, a real-time news feed for sharing posts with images and text, and a complete friend system with requests and connections. It's a comprehensive demonstration of building a full-stack application using Java for both the frontend and backend.
